\# AI Phishing Email Detector



\## Overview

This project is a Machine Learning based phishing email detection system developed using Python and Scikit-learn.



The system analyzes email text and classifies emails as:

\- Safe

\- Phishing



\## Features

\- Machine Learning based detection

\- TF-IDF feature extraction

\- Logistic Regression classifier

\- GUI using Tkinter

\- Confusion Matrix visualization

\- URL and keyword analysis

\- High accuracy (\~96%)



\## Technologies Used

\- Python

\- Pandas

\- Scikit-learn

\- Tkinter

\- Matplotlib

\- Seaborn



\## How to Run

1\. Install required libraries

2\. Place emails.csv in project folder

3\. Run:



```bash

python main.py

```



\## Output

The application detects phishing emails and displays:

\- Prediction result

\- Accuracy

\- Suspicious keywords

\- URL analysis

\- Confusion matrix
